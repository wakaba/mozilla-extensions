CSS Reference Sidebar

この Sidebar panel は、 CSS の仕様書等の特定の属性などの説明をすぐに読めるようにするための栞のようなものです。

同様のものとして DevEdge Sidebars: CSS2 Quick Reference にも CSS 2, CSS 2.1 用がそれぞれありますが、こちらは CSS 1 と CSS 2 へのリンクを同時に表示すると共に、 CSS Laboratory の対応表・ノートやブラウザの仕様説明なども含めています。 (CSS 2.1, CSS 3 はいずれ追加します。仕様書以外のものは特に参考になると思われるものを選んで追加しています。)

構成ファイルの入手

単に Sidebar に追加するだけの場合は先程のボタンを押すだけで結構です。 再利用する場合などは次の資源をお持ち帰り下さい。

制限事項

  • 一部 reference では文書内に適切な ID が振られていなかったために直接当該属性等の説明にまで飛べないことがあります。 (出来るだけ近くに飛びます。)
  • mso-* 属性など Microsoft Office の拡張はあまり含んでいません。 これは、その仕様書, リファレンスの類が無く、全体像もはっきりしないためです。 (少なくても60個の独自拡張属性・記述子が存在していることが分かっています。)
  • 独自拡張属性等を表示しない簡単な方法は今のところありません。
  • 長い属性等名のため Sidebar の幅と文字の大きさの設定によっては横 scroll bar が出ます。 (Mozilla の仕様)
